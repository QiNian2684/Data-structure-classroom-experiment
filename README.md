# 数据结构课堂实验

本仓库包含软件工程专业第3学期“数据结构”课程的实验代码。所有代码均使用C++编写，旨在提供实验课所需的所有代码实现，包括普通实验和扩展实验。

## 实验目的

本课程的实验旨在帮助学生深入理解数据结构的原理，并通过编写C++代码来加强对数据结构在实际应用中的理解和运用。

## 实验内容

### 普通实验

- **实验1**：介绍基础数据结构的实现，如链表、栈和队列。
- **实验2**：深入探讨树结构，包括二叉树、二叉搜索树和平衡树的实现及操作。
- **实验3**：研究图结构，包括图的表示和基础算法如深度优先搜索和广度优先搜索。
- **实验4**：探索更复杂的数据结构，如优先队列、堆以及各种排序算法。

### 扩展实验

- **基于Dijkstra算法的重庆邮电大学最短路径求解**：使用Dijkstra算法求解重庆邮电大学校园地图的最短路径问题。
- **基于哈夫曼树的英文文本数据压缩算法**：实现哈夫曼编码算法对英文文本数据进行压缩，探讨其在数据压缩中的应用。

## 如何使用

要运行这些实验代码，确保您的系统中已经安装了C++编译器如g++。

```bash
# 克隆仓库
git clone https://github.com/yourusername/data-structure-classroom-experiment.git

# 进入相应实验目录
cd data-structure-classroom-experiment/experiment1

# 编译代码
g++ -o experiment1 experiment1.cpp

# 运行编译后的程序
./experiment1
